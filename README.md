# omniverse-playground

## Introduction

This project is designed to provide hands-on experience with the NVIDIA Omniverse / Isaac Sim technology stack.
It documents the learning journey, tackling each step from a beginner's perspective.

## Installation

[NVIDIA Isaac Sim — Workstation Installation Guide](https://docs.isaacsim.omniverse.nvidia.com/latest/installation/install_workstation.html)

## Requirements

| Item | Spec |
|------|------|
| GPU | NVIDIA RTX A6000 48GB |
| OS | Ubuntu 22.04 |
| Isaac Sim | 5.1.0 |
